STEM Curriculum

A comprehensive year-long journey through Science, Technology, Engineering, and Mathematics. Students progress from basic 3D design to advanced robotics and game development.

Year-Long Learning Journey

Our curriculum is designed to build skills progressively, with each quarter focusing on different aspects of STEM while maintaining connections between subjects.

Quarter 1

3D Design & Treehouses

Tinkercad Mastery

Students begin their STEM journey with 3D design using Tinkercad. They learn fundamental engineering principles while creating architectural models, treehouse designs, and problem-solving solutions.

Key Learning Objectives:
  • Spatial reasoning and 3D visualization
  • Basic engineering principles
  • Problem-solving through design
  • Digital literacy and tool mastery
Skills Developed:
3D Modeling Architecture Engineering Problem Solving

Quarter 2

Game Development

Scratch Programming

Building on spatial skills, students dive into programming logic using Scratch. They create interactive games, stories, and animations while learning computational thinking.

Key Learning Objectives:
  • Computational thinking and logic
  • Sequential and conditional programming
  • Creative storytelling through code
  • User experience design
Skills Developed:
Programming Game Design Logic Storytelling

Quarter 3

Unreal Engine

3D Environment Creation

Students advance to professional-grade 3D environments using Unreal Engine. They create immersive worlds, interactive experiences, and learn industry-standard tools.

Key Learning Objectives:
  • Advanced 3D environment design
  • Interactive experience creation
  • Professional tool proficiency
  • Collaborative project development
Skills Developed:
3D Environments Game Engines Interactive Design Professional Tools

Quarter 4

Robotics

Physical Computing

Students bring their digital skills into the physical world through robotics. They build, program, and control robots while solving real-world challenges.

Key Learning Objectives:
  • Physical computing and sensors
  • Mechanical engineering basics
  • Real-world problem solving
  • Team collaboration and project management
Skills Developed:
Robotics Physical Computing Mechanical Engineering Teamwork

Learning Progression

Each quarter builds upon previous learning while introducing new challenges and technologies.

Foundation Building

Students develop spatial reasoning and basic engineering principles through hands-on 3D design.

Logic & Programming

Building on spatial skills, students learn computational thinking and programming logic.

Advanced Applications

Students apply their skills to create professional-grade 3D environments and experiences.

Real-World Integration

Students bring digital skills into the physical world through robotics and engineering challenges.

Assessment & Standards

Our curriculum aligns with national STEM standards and focuses on authentic assessment through project-based learning.

Project-Based Learning

Students demonstrate learning through authentic projects that solve real problems and showcase creativity.

Progressive Assessment

Continuous assessment through portfolios, peer review, and teacher feedback ensures growth and understanding.

Standards Alignment

Curriculum aligns with NGSS, ISTE, and Common Core standards for comprehensive STEM education.
